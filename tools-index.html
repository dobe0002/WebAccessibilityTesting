Tools

Free tools to assist with web accessibility testing

Browser Plugins/Add-ons/Extensions

Automated checking

Tools that check for a wide variety of issues.

  • AInspector (Firefox) (requires Firefox ESR) (new window)
  • WAVE (Chrome and Firefox) (new window)
  • aXe (Chrome and Firefox) (new window)

Specific issue checking

Used for manual checks.

  • WAVE (Chrome and Firefox) (new window)
  • W15YQC (Firefox) (requires Firefox ESR) (new window)
  • High Contrast (Chrome) (new window)
  • No Coffee Vision Simulator (Chrome) (new window)
  • ColorA11y (Chrome) (new window)
  • Web Developer Toolbar (Chrome and Firefox) (new window)
  • Contrast Ratio Checker (Chrome) (new window)

Bookmarklets

Bookmarks that assist in performing manual checks.

  • Tota11y (new window)
  • SkipTo (new window)
  • Accessibility Bookmarklets (new window)

Screen Readers

  • NVDA (Windows) (new window)
  • JAWS (Windows) (new window)
  • VoiceOver(Mac) (new window)
