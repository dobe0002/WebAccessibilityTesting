Spacing

Things to check for:

    Content is still visible/available when:

  • Line height is set to 1.5 times the font size.
  • Paragraph spacing is set to 2 times the font size.
  • Letter spacing is set to 0.12 times the font size.
  • Word spacing is set to 0.16 times the font size.
Information

Use the following CSS to check spacing:

* { line-height:1.5em !important} /* Line spacing */

p {margin-bottom:2em !important} /* Paragraph spacing */

* {letter-spacing: 0.12em !important} /* Letter spacing */

* {word-spacing: 0.16em !important;} /* Word spacing */
