Tools

Free tools to assist with web accessibility testing

Browser Plugins/Add-ons/Extensions

Automated checking

Tools that check for a wide variety of issues.

  • WAVE (Chrome and Firefox)
  • aXe (Chrome and Firefox)
  • Site Improve (Chrome and Firefox)
  • Lighthouse (Chrome)
  • FAE (online)

Specific issue checking

Used for manual checks.

  • ColorA11y (Chrome)
  • Web Developer Toolbar (Chrome and Firefox)
  • User CSS (Chrome)
  • High Contrast (Chrome)
  • No Coffee Vision Simulator (Chrome)
  • Contrast Ratio Checker (Chrome)

Bookmarklets

Bookmarks that assist in performing manual checks.

  • Tota11y
  • Text Spacer
  • Autocorrect Identifier
  • Accessibility Bookmarklets

Screen Readers

  • NVDA (Windows)
  • JAWS (Windows)
  • VoiceOver(Mac)
  • ChromeVox (Chrome extension)
Tools referenced in this document are for informational purposes only. Neither Kim nor the University of Minnesota officially endorse any of the tools mentioned in this site.
