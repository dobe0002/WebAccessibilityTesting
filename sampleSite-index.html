Sample Pages

This mini-site contains pages for demonstrating and practicing web accessibility testing.

Because this is a "demo" site, the pages are "rough", have accessibility errors, and contain nonsensical text.

How to test document (new window)

Testing notes

There are a couple of known "false positives" with this site.

  • In WAVE, there is an error for a "Broken skip link". This link does function correctly.
  • In WAVE, there is an alert for an "Accesskey". This accesskey does not interfere with accessibility.
  • In aXe, there are two color contrast errors about the page header. This isn't an issue.
